This directory contains the source files for QtlMovie and dependent libraries.
Subdirectories:
  libdvdcss - 3rd party open-source library to read encrypted DVD's
  libQtl    - Qt utility library
  libQts    - Qt MPEG Transport Stream library
  QtlMovie  - Application GUI, based on Qtl and Qts
  TestUnit  - Automated unitary tests
  TestTool  - Command line test tool for various modules
