Overview

QtlMovie is a simple Qt front-end to ffmpeg and other media tools.

QtlMovie does not pretend to be the next most wanted GUI for ffmpeg. Instead, it is designed to automate a reduced number of tasks in a "one-click" way without setting many parameters.

These tasks are:

  • Create a simple video DVD with one movie, no menu, no alternate language.
  • Create video files for the iPad.

QtlMovie main features are:

  • Take any type of common audio/video file as input.
  • Produce ISO images for DVD burning (external tool required to burn the ISO image).
  • Produce video files for iPad viewing.
  • Keep only one audio track.
  • Optionally hardcode ("burn") one subtitle track into the video but does not keep subtitles as a separate stream.

QtlMovie is based on the Qt framework. It works on Windows, Linux and should work on any platform which is supported by Qt version 5. QtlMovie uses external free software such as ffmpeg, DVD Author or mkisofs which are available on Windows and Linux.

QtlMovie is available either as source code or binary installer for Windows. The binary installer for Windows comes prepackaged with Qt5 libraries, ffmpeg (64-bit), DVD Author and mkisofs. It is self-sufficient and does not depend on any prior installation.

Input files

Audio/video input

Input files can be any type of multimedia file which is accepted as input by ffmpeg. This include the container file format (MPG, AVI, MKV, etc.), the video format (MPEG-2, AVC / H.264, WMV, etc.) and the audio format (MP2, MP3, AAC, etc.)

Multiple video, audio and subtitle streams can be present. The QtlMovie graphical interface requests the user to select exactly one video and one audio stream.

Subtitle input

At most one subtitle source can be specified. This can be a stream of the input file or an external subtitle file.

The following subtitles formats are supported:

  • SubRip (SRT).
  • SubStation Alpha (SSA, ASS).
  • DVD subtitles (input file stream only, not external subtitle file).
  • DVB subtitles (transport stream input file stream only, not external subtitle file).

Teletext subtitles are not yet supported but will be in a future release.

Important: The subtitles are always hardcoded ("burnt") into the video by QtlMovie. The output file never contains independent subtitle streams.

Output files

DVD output

QtlMovie always produces DVD with the following characteristics:

  • PAL format.
  • MPEG-2 video, 16:9, 720 x 576, 25 frames/second. The average bitrate is 4 Mb/s (adjustable in the settings) but is automatically reduced when necessary so that the output fits in a 4.7 GB DVD media.
  • MPEG-1 layer 2 audio, stereo, sampled at 48 kHz. The audio bitrate is 192 kb/s.

The output video has a 16:9 aspect ratio. Black borders are added when necessary so that the meaningful image part has the same aspect ratio as the input file.

QtlMovie can produce two types of DVD output: An MPEG file (.mpg) with a DVD-compliant content or an ISO image of the DVD media. The first option can be used to check the audio/video using standard media players. If the result is satisfactory, the resulting MPEG file can be used as input to produce a DVD ISO image. Alternatively, if you are confident with the processing, a DVD ISO image can be directly requested (the transcoding and ISO image generation are automatically processed).

When producing a DVD ISO image, QtlMovie tries to guess if the input file is already a DVD-compliant MPEG file or not. In the first case, QtlMovie does not re-encode the input file and directly creates the DVD structure from the input MPEG file.

iPad output

QtlMovie always produces iPad files with the following characteristics:

  • MP4 file format.
  • AVC / H.264 video, baseline profile, level 3.0. The average bitrate is 2.5 Mb/s. The display aspect ratio is unchanged (no black border is added). The frame size is unchanged, unless the input size exceeds 1024 x 768 in which case the size is reduced to fit within a 1024 x 768 bounding box, preserving the input aspect ratio.
  • AAC audio, stereo, sampled at 48 kHz. The audio bitrate is 160 kb/s.

Subtitle extraction

QtlMovie can also be used to extract subtitle streams from the input files. The input and output formats must text subtitles: SRT, SSA, ASS. Conversions are possible between these formats (SRT subtitles can be extracted as ASS for instance).
